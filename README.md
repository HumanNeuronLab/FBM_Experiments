## ▶️ How to Run the Experiment  

0. Connect the **photodiode** to the acquisition system and attach it to the **top-left corner** of the experiment laptop.  
1. Double-click the respective executable file:  
   - **Category Localizer:** `CL_....exe`  
   - **Functional Mapping:** `LM_....exe`  
2. A command window will appear. The experiment should begin after a few seconds.  
3. Fill in the required fields in the experiment’s pop-up window, then press **OK**.  
4. Run the experiment until completion.  
5. After finishing, double-check the **behavioral file** inside the `Results` folder.
